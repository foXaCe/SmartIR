# Security Policy

## Supported versions

Seule la dernière version publiée sur la branche `main` est supportée.

## Reporting a vulnerability

**N'ouvrez pas d'issue publique** pour une vulnérabilité de sécurité.

Merci de la signaler de façon responsable via un
[private vulnerability report](https://github.com/foXaCe/SmartIR/security/advisories/new).

Délai de réponse cible : 7 jours.
