# SmartIR

Control IR/RF devices — air conditioners, fans, TVs and lights — through Broadlink,
Xiaomi, LOOKin, ESPHome and MQTT controllers, directly from Home Assistant.

## Features

- Climate, fan, media player and light control over IR/RF
- Config flow UI setup
- State restoration and power-sensor sync

## Configuration

**Settings** → **Devices & Services** → **Add Integration** → "SmartIR".
